# Simulador Universe

Simulación gravitacional 3D desarrollada en Python utilizando Ursina Engine.

## Características

- Simulación N-Body
- Atracción gravitacional entre partículas
- Fusión de partículas al colisionar
- Conservación del momento lineal
- Visualización 3D interactiva

## Tecnologías

- Python
- Ursina Engine

## Cómo ejecutar

pip install ursina

python universe.py

## Futuras funciones
- Colores según masa
    - Pequeñas = azules
    - Medianas = amarillas
    - Grandes = rojas
- Trails (estelas) para ver las órbitas.
- Temperatura
Cada choque genera calor.
- Explosiones
  - Si una partícula supera cierta masa.
- Tipos de partículas
  - Tipo A + Tipo A → Tipo B
  - Tipo B + Tipo C → Tipo D
