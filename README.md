# Simulador Universe

Simulación gravitacional 3D desarrollada en Python utilizando Ursina Engine.

## Características

- Simulación N-Body
- Atracción gravitacional entre partículas
- Fusión de partículas al colisionar
- Conservación del momento lineal
- Visualización 3D interactiva

## Tecnologías

- Python
- Ursina Engine

## Cómo ejecutar

pip install ursina

python universe.py
